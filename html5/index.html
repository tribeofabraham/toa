NCTM Pandemic

This is a remake of an app I had created many years ago which for obvious reasons suddely became relavent again. In the app I simulated Fauci's simple three variable equation to show not only the random variability of the outcomes but also how moving a data point like "contacts" could make a big difference.

There is an accompanying lesson plan but you can just hit calculate then play. The SVG files are rendered dynamically within the DOM.

If you rollover just below and to the left of the anniversary logo until your cursors disappears and then click you will unlock an easter egg which allows larger populations

The bar chart representing the data is rendered as SVG from within the DOM. Every time "Calculate" is clicked a new random set which is used to create the next plot.

HTML5, SVG, Procedural Animation

Letter Tiles

Client had an interactive that accompanied their classroom instructions and materials

The bar for WCAG and ADA is much higher now in the academic community and they asked me to help them reimagine this tool as a compliant HTML5 asset.

Building an asset like this dynamically from JSON data allows me to address WCAG and accessibility issues BEFORE the asset is dropped in the DOM. Loading the tabs data from a JSON source allows the tile sets and instructional videos to be easily customized. There is a feature that allows an alternate tab set to be loaded via URL parameters.

HTML5, JSON, Streaming Media, WCAG

Nashville Notation

An app in progress for Ragamuffin Hall to introduce fans to a method of communicating music through numbers instead of notes.

Choosing a key and a progression will reveal the chord patterns. Transpose will raise or lower the pitch of the progression relatively while swapping the visible layers of the single SVG chord chart.

I am fascinated with the music and science of music and also program IoT hardware devices for sequencing and sampling.

HTML5, SVG
